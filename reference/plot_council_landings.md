# Plotting landings proportions by ASMFC Council Management Zones

Function to plot proportions of landings across Council Management Zones
for Mid-Atlantic species.

## Usage

``` r
plot_council_landings(species = "all", data = "landings")
```

## Arguments

- species:

  Mid-Atlantic managed species as listed in \`species_list(source =
  "landings")\`

- data:

  Landings outputs from \`pull_garfo_landings()\`

## Value

List of faceted plots.
