# Pull federal VTR data

Function to pull and clean Vessel Trip Reports from pre-existing
confidential repository.

## Usage

``` r
pull_vtr(proj_path)
```

## Arguments

- proj_path:

  Local path to data file

## Value

Data frame of vessel trip reports; includes year, sub_trip_id, latitude,
longitude, port name and state, species caught, weight of kept and
discarded catch.
