# Plotting landings trends

Function to plot landed value and volume trends for Mid-Atlantic
species.

## Usage

``` r
plot_landings_trends(species = "all", data = "landings")
```

## Arguments

- species:

  Mid-Atlantic managed species as listed in \`species_list(source =
  "landings")\`

- data:

  Landings outputs from \`pull_garfo_landings()\`

## Value

List of faceted plots.
