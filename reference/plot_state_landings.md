# Plotting landings proportions by state

Function to plot proportions of landings across states for Mid-Atlantic
species.

## Usage

``` r
plot_state_landings(species = "all", data = "landings")
```

## Arguments

- species:

  Mid-Atlantic managed species as listed in \`species_list(source =
  "landings")\`

- data:

  Landings outputs from \`pull_garfo_landings()\`

## Value

List of faceted plots.
